How to use
You only need to set this up once when using the extension for the first time.
1 Choose an AI provider and save its API key.
Gemini: aistudio.google.com/apikey
Claude: console.anthropic.com
OpenAI: platform.openai.com/api-keys
2 Click a kanji on a Japanese page to open a context-aware yomigana popup.
3 최근 7일간 2회 이상 조회한 항목은 🔥 Focus에 자동으로 모입니다.
4 For text inside links or buttons, use Alt-click (Option-click on Mac) to force a lookup.
API providers
Pick the model you want to use and save its API key. The popup never shows the full key again, only a masked suffix.
